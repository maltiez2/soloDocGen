# soloDocGen
Generation documentation for solo

## Requirements(# pip install):
   - treelib
   - PySide2
    
## Usage:
   - **Enter to run documentation viewer:**
    ```
    ./soloDocGen.py -p PATH
    ```
   - **Where:**
     - PATH : path to 'core' folder
   - **Options:**
     - -p : path to 'core' folder
